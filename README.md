# 📱 Projeto iPhone - UML

Este projeto implementa um modelo orientado a objetos baseado no diagrama UML fornecido. Ele simula um **iPhone** com funcionalidades de:

- **Reprodutor Musical**
- **Aparelho Telefônico**
- **Navegador de Internet**
